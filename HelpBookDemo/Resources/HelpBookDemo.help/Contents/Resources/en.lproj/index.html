Welcome to HelpBookDemo Help

This is a simple help book page for demonstration purposes.

Try modifying the content and building the project.

The next time you run the project, you should see the updated help page. This works because the build phase "Clear Help Cache" deletes the previous cache file from

~/Library/Group Containers/group.com.apple.helpviewer.content/Library/Caches/

Without this step, Apple's' Help Viewer app (Tips.app) will never show your updates.
